# Strava-challenge

## Getting Started

```bash
# Install required packages
$ yarn

# Run the development server
$ yarn dev
```

The dev server starts on [http://localhost:3000](http://localhost:3000)

## Code formatting

Use the following commands to ensure your code meets our coding standards:

```bash
$ yarn prettier
$ yarn lint
```
